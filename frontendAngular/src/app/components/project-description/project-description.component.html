Gerador de Senhas

Aqui você gera senhas fortes e seguras!

Para gerar sua senha clique em gerar nova senha!

Como utilizar?

O gerador de senhas cria automaticamente senhas fortes e aleatórias com base nas configurações preferenciais. É possível ajustar o número de caracteres e os tipos de caracteres necessários para a senha (letras minúsculas e maiúsculas, números e símbolos). Após a seleção, basta clicar em gerar nova senha.

Sobre o projeto

Tecnologias Utilizadas:
Backend: Java 16
A lógica de geração de senhas é implementada utilizando Java 16, que traz melhorias significativas de desempenho e novas funcionalidades. O backend é responsável por receber as solicitações de geração de senhas e fornecer uma API RESTful para o frontend. Utilizamos bibliotecas de segurança e algoritmos de geração aleatória para garantir a criação de senhas fortes e seguras, adaptadas às preferências do usuário, como comprimento da senha e inclusão de caracteres especiais.

Frontend: Angular 18
O frontend é desenvolvido em Angular 18, permitindo uma interface de usuário responsiva e dinâmica. Os usuários podem facilmente interagir com a aplicação para gerar senhas, selecionar critérios (como comprimento, uso de caracteres especiais, etc.) e visualizar o histórico de senhas geradas. A comunicação entre o frontend e o backend é feita por meio de serviços HTTP, facilitando a troca de dados e a atualização em tempo real do histórico de senhas.

Funcionalidades:
Geração de Senhas Aleatórias: Os usuários podem personalizar suas senhas, escolhendo o comprimento desejado e se desejam incluir letras maiúsculas, minúsculas, números e caracteres especiais.

Histórico de Senhas: A aplicação mantém um registro de todas as senhas já geradas pelo usuário. Isso permite fácil acesso a senhas anteriores, promovendo conveniência e segurança.

Interface Intuitiva: A interface do usuário é projetada para ser simples e direta, permitindo que usuários de todos os níveis de experiência naveguem pela aplicação sem dificuldades.

Segurança Aprimorada: Utilizando práticas de segurança recomendadas, como a geração de senhas fortes e a proteção dos dados dos usuários, o aplicativo assegura que as informações sejam mantidas seguras.

Conclusão:
O Gerador de Senhas é uma ferramenta útil e eficaz para quem busca uma solução prática e segura para a criação de senhas. Com a combinação do poder do Java 16 no backend e a modernidade do Angular 18 no frontend, este projeto oferece uma experiência de usuário excepcional e um nível elevado de segurança.
